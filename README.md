# Student Medical Health System (IT2234 ICA3)

A full-stack medical appointment scheduling web application designed to modernize university healthcare triage, eliminating administrative bottlenecks and manual record vulnerabilities using the MERN stack.

---

## 🚀 Key Features
- **Symmetrical 3-Column Interface:** Clean, accessible frontend form for students to submit registration profiles and medical reasons.
- **Asynchronous Data Triage:** Pre-appointment data intake allowing clinical staff to prioritize high-severity cases.
- **Mongoose Schema Security:** Enterprise-grade data validation preventing faulty database entry strings or missing student inputs.
- **Cross-Device Fluidity:** Responsive web layouts powered by CSS Grid and Flexbox for mobile and desktop access.

---

## 🛠️ Tech Stack & Mechanisms Used
- **Frontend:** React.js, Axios, CSS3 (Flexbox/Grid, Media Queries)
- **Backend:** Node.js, Express.js (REST API Architecture, CORS Security Middleware)
- **Database:** MongoDB Cloud Clusters via Mongoose ODM
- **Environment Management:** Dotenv (Operational port variables and database string insulation)
